# 括号生成                                                           

* ID: 22      | Passing rate: 77.5% | PaidOnly: false  | Difficulty: Medium 
* Url: https://leetcode.cn/problems/generate-parentheses/ 
* Topic: 字符串, 动态规划, 回溯 

## Test Case: 
```
3
1
```



---
数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且
**有效的 **括号组合。


**示例 1：**

**输入：**n = 3**输出：**[\"((()))\",\"(()())\",\"(())()\",\"()(())\",\"()()()\"
]

**示例 2：**

**输入：**n = 1**输出：**[\"()\"]


**提示：**

* `1 <= n <= 8`
