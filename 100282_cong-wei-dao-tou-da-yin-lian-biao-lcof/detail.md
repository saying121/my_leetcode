# 从尾到头打印链表                                                       

* ID: 100282  | Passing rate: 74.3% | PaidOnly: false  | Difficulty: Easy 
* Url: https://leetcode.cn/problems/cong-wei-dao-tou-da-yin-lian-biao-lcof/ 
* Topic: 栈, 递归, 链表, 双指针 

## Test Case: 
```
[1,3,2]
```



---
输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。


**示例 1：**

**输入：**head = [1,3,2]**输出：**[2,3,1]


**限制：**

`0 <= 链表长度 <= 10000`
